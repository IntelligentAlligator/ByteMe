# Digital Cookbook

A simple React web app to browse, view, and search recipes. Recipes are stored in JSON for easy editing. Ready for deployment to GitHub Pages.

## Features
- Browse all recipes
- View recipe details
- Search recipes by title or ingredient
- Recipes stored in `recipes.json`

## Getting Started
1. Install dependencies:
   ```sh
   npm install
   ```
2. Start the development server:
   ```sh
   npm run dev
   ```
3. Build for production:
   ```sh
   npm run build
   ```
4. Preview production build:
   ```sh
   npm run preview
   ```

## Deployment
- Configure for GitHub Pages by updating `vite.config.js` with the correct `base` path.
- Deploy the `dist` folder to GitHub Pages.

## Editing Recipes
- Edit or add recipes in `recipes.json`.
